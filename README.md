# Attendance Buddy

A mobile-first attendance calculator with predictions and analytics.

## Features

- Track your attendance (present/total classes)
- Calculate attendance percentage with visual progress circle
- Predict how many classes you need to attend to reach your target
- Configurable required attendance percentage
- Dark/Light theme toggle
- Works as a native Android app via Capacitor

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **Capacitor** for native Android builds

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Build Android APK
npx cap sync
npx cap open android
```
